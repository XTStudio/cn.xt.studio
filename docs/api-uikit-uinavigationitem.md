```UINavigationItem``` 配合视图控制器使用，用以描述导航信息。

## API

### Instance Fields

#### title: `string`
导航栏标题。

#### hidesBackButton: `boolean`
是否隐藏导航栏上的后退按钮。

#### leftBarButtonItem: `UIBarButtonItem`
导航栏左侧按钮。

#### leftBarButtonItems: `UIBarButtonItem[]`
导航栏左侧多个按钮。

#### rightBarButtonItem: `UIBarButtonItem`
导航栏右侧按钮。

#### rightBarButtonItems: `UIBarButtonItem[]`
导航栏右侧多个按钮。
