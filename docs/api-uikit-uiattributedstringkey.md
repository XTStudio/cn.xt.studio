title: UIAttributedStringKey
---

描述具体的富文本样式。

```typescript

// 文字颜色。
foregroundColor              -> UIColor

// 字体。
font: string                 -> UIFont

// 背景色。
backgroundColor: string      -> UIColor

// 字距。
kern: string                 -> number

// 该值为 1 时，显示删除线。
strikethroughStyle: string   -> number

// 该值为 1 时，显示下划线。
underlineStyle: string       -> number

// 描边颜色。
strokeColor: string          -> UIColor

// 描边线条宽度。
strokeWidth: string          -> number

// 下划线颜色。
underlineColor: string       -> UIColor

// 删除线颜色。
strikethroughColor: string   -> UIColor

// 段落样式。
paragraphStyle: string       -> NSParagraphStyle
```
