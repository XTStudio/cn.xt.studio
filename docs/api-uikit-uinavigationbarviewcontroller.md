创建一个自带导航栏的视图控制器。

## Extends

[UIViewController](./api-uikit-uiviewcontroller.md)

## API

### Instance Fields

#### navigationBarContentHeight: `number`
指定导航栏的高度。

#### navigationBarInFront: `boolean`
指定导航栏是否位于视图层级的最高级。

#### navigationBar: `UIView`
返回导航栏视图实例。
