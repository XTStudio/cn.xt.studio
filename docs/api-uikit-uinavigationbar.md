[UIView](./api-uikit-uiview.md)

## API

### Instance Fields

#### translucent: `boolean`
导航栏是否为透明样式。

#### barTintColor: `UIColor | undefined`
导航栏的背景色。

#### titleTextAttributes: { [key: UIAttributedStringKey]: any }
标题样式。

#### backIndicatorImage: `UIImage | undefined`
使用一张自定义图片替换默认后退按钮。

#### backIndicatorTransitionMaskImage: `UIImage | undefined`
使用一张自定义图片替换默认后退按钮。
