title: UIParagraphStyle
---

用于描述富文本的段落样式。

## API

### Instance Fields

#### lineSpacing: `number`
行距。

#### alignment: `UITextAlignment`
文本对齐方式。

#### lineBreakMode: `UILineBreakMode`
断行方式。

#### minimumLineHeight: `number`
最小行高。

#### maximumLineHeight: `number`
最大行高。

#### lineHeightMultiple: `number`
行高。
