```UITabBarItem``` 配合视图控制器使用，用以描述 Tab 信息。

## API

### Instance Fields

#### title: `string | undefined`
Tab 标题。

#### image: `UIImage | undefined`
Tab 图标（正常状态）。

#### selectedImage: `UIImage | undefined`
Tab 图标（选中状态）。

#### imageInsets: `UIEdgeInsets`
图标的布局调整值。
