使用 ```CAGradientLayer``` 渲染一个渐变图层。

## Extends

[CALayer](./api-coregraphics-calayer.md)

## API

### Instance Fields

#### colors: `UIColor[]`
渐变色列表。

#### locations: `number[]`
各渐变色的位置。

#### startPoint: `UIPoint`
渐变色的起始位置，取值范围为 `[0.0, 1.0]`。

#### endPoint: `UIPoint`
渐变色的结束位置，取值范围为 `[0.0, 1.0]`。
